# 「WHUT」Android客户端

### 武汉理工大学教务处图书馆查询软件

此软件可方便学生查询课表以及图书借阅查询等，于2012年12月24日发布[初版](http://www.withparadox2.com/archives/80)，现在准备重新编写此软件。由于教务处改版，目前主要致力于图书馆功能的开发。</br>

### 已经具备的功能

1. 查询课表以及离线课表
2. 查询图书借阅以及离线借阅
3. 图书续借
4. 图书搜索
5. 要借图书

## 常见问题

### 这款软件是否安全

所有数据都是通过链接到学校相关网站来获取的，相关信息都记录在手机本地，不会有任何信息泄露，这点大可放心。再说，我要你们信息干什么呢？

### 这款软件是否免费

当然免费，要知道免费都没人用呀~~

没有！写此软件纯粹是个人兴趣，不可以侮辱爱好，兴趣是无价的:-)
